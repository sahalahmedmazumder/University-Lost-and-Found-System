# University Lost and Found System

**Description**

A web-based application developed to help students report lost items, publish found items, search records, and recover belongings through a centralized university platform.

 **Features**

* User Registration and Login
* Report Lost Items
* Report Found Items
* Search and Filter
* Dashboard
* Record Management

## **Technology Stack**

Frontend:

*React*

Backend:

*FastAPI*

Database:

*PostgreSQL*
