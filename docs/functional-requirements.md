# Functional Requirements

**FR-01 User Registration**

The system shall allow users to create an account.

*Inputs:*

Name
Email
Password

*Output:*
Successful registration

**FR-02 User Login**
The system shall authenticate users.

*Output:*
Dashboard access

**FR-03 Report Lost Item**
The system shall allow users to submit lost item reports.

*Inputs:*
Item Name
Description
Category
Location

**FR-04 Report Found Item**
The system shall allow users to submit found item reports.

**FR-05 Search Items**

The system shall allow users to search items.

**FR-06 Update Item Status**

The system shall allow administrators to update status.

**FR-07 Dashboard**

The system shall display summary information.

**FR-08 Manage Records**

The administrator shall manage reports.
