# Product Requirement Document (PRD)

**Executive Summary**

The University Lost and Found Management System is a web-based platform that helps students report lost items, publish found items, search records, and recover belongings through a centralized system.
The project aims to improve item recovery efficiency and reduce dependence on manual communication.

**Problem Statement**

Current lost and found processes are informal and difficult to manage.

Common problems:
Scattered communication
Difficult item tracking
Delayed recovery
Missing records
Proposed Solution

Develop a centralized web application with:
Authentication
Lost item reporting
Found item reporting
Search and filter
Dashboard
Item status tracking

**Scope**

**Included**
Register/Login
Report management
Search functionality
Dashboard
Database storage

**Excluded**

Payment
Chat
Mobile app
Notifications

**Business Goals**
Improve recovery process
Reduce manual work
Organize records

**Success Metrics**

Metric	Target

*Login Success	>90%*
*Search Response	<3 sec*
*Report Completion	>85%*
*CRUD Accuracy	100%*
