# University Lost and Found Management System

**Introduction**

The University Lost and Found Management System is a web-based application designed to simplify and organize the process of reporting and recovering lost items within a university campus. The system provides a centralized platform where students can report lost belongings, publish found items, search existing records, and track item recovery status.

The application replaces manual and scattered reporting methods with an organized digital solution.

**Background**

Universities frequently experience situations where students lose personal belongings such as ID cards, wallets, keys, books, electronic devices, and accessories.

Currently, recovery depends on:

Social media posts
Informal communication
Student communities
Manual announcements

These methods are often inefficient and difficult to track.

A centralized platform can improve communication and increase successful item recovery.

**Project Goal**

The goal of this project is to design and develop a full-stack web application that improves campus lost and found operations.

**The system aims to:**

Provide centralized reporting
Improve item recovery efficiency
Reduce manual record handling
Create a better user experience
Objectives
Primary Objectives
Develop a digital platform for lost and found reporting.
Allow users to search and view item records.
Enable administrators to manage reports.
Store records securely in a database.
Improve communication between users.

Students

Report found items
Search records
Track item status
Administrators

**Administrators will be able to:**

Monitor reports

Update item status

**Expected Outcome**

The expected outcome is a responsive and user-friendly web application that provides a structured process for reporting and recovering lost belongings inside the university.

The application should improve accessibility, reduce recovery time, and maintain organized records.
