# Survey Report


**Objective**
Measure interest in a university lost and found system.

**Survey Questions**

Have you lost items on campus?
Have you found items but did not know where to report?

Would you use a lost and found website?

### Sample Results

| Question | Yes | No |
| :--- | :--- | :--- |
| Lost item before | 75% | 25% |
| Found item before | 68% | 32% |
| Would use system | 90% | 10% |


**Analysis**
Survey responses indicate strong demand for a centralized reporting platform.
Most students prefer a digital solution instead of informal communication channels.


**Conclusion**

Survey findings support the feasibility of implementing the system.
