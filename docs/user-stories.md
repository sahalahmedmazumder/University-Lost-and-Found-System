# User Stories

**Authentication**

*As a student,*
I want to register an account,
So that I can access the system.

*As a user,*
I want to login,
So that I can manage reports.

**Item Reporting**

*As a student,*
I want to report lost items,
So that others can identify them.

*As a student,*
I want to report found items,
So that owners can recover belongings.

**Search**

*As a user,*
I want to search items,
So that I can locate records quickly.

*Administration*

As an administrator,
I want to manage reports,
So that records remain organized.
